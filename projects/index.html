Concurrent Biological Sequence Processing with Go

Many sequence manipulations are "embarrassingly parallelizeable" and Go makes concurrency incredibly simple. As part of our machine learning efforts in the Armbrust Lab I have been developing a number of preprocessing tools for DNA/RNA sequence manipulation.

GoMrLBP

Inspired by Kouchaki et al. (2019), I implemented multi-resolution local binary patterns in Go. Beyond the original C++ implementation, the Go implementation can run on multiple cores and enables large volumes of sequence to be processed in seconds to minutes.
