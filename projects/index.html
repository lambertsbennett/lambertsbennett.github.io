Concurrent Biological Sequence Processing with Go

Many sequence manipulations are "embarrassingly parallelizeable" and Go makes concurrency incredibly simple. As part of our machine learning efforts in the Armbrust Lab I have been developing a number of preprocessing tools for DNA/RNA sequence manipulation.

GoMrLBP

Multi-resolution local binary patterns for biological sequence data in Go. Inspired by Kouchaki et al. (2019), I implemented multi-resolution local binary patterns in Go. Beyond the original C++ implementation, the Go implementation can run on multiple cores and enables large volumes of sequence to be processed in seconds to minutes.



GoKmer

Concurrent k-mer expansion of biological sequence data in Go. When pairing NLP techniques with biological sequence data, a frequent operation is to explode a sequence into its constitutive k-mers. This can be a very slow and memory intensive operation if you're not careful. Here I implemented a very simple tool in Go that concurrently expands a sequences into k-mers. Large assemblies can be processed on the order of seconds to minutes on an 8 core laptop.
